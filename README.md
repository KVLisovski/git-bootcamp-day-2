# git-bootcamp-day-2
Репозиторий для буткэмпа слерм день 2.
Это проект автоматической загрузки сертификатов сотрудников в СЭД. Для подписния документов. Проверяет дату актуальноси в СЭД и Active Directory.

## Быстрый старт

```bash
git clone git@github.com:KVLisovski/git-bootcamp-day-2.git
cd /git-bootcamp-day-2
java -jar load_certs.jar --help
```

## Установка

Необходимо использование JAVA версии 17 или выше. Установить пароль для подклбючения к AD (Опуианально для тестирования передать в переменную $PASSWORD)

```bash
java --version
PASSWORD="ваш_пароль"
```

## Использование

Простейший сценарий для проверки доступа к API:

```bash
java -jar load_certs.jar --host="ваш хост" --test-api
# → Запуск тестрирования API
# → Хост доступен
# → GET - Success!
# → POST - Success!
# → PUT - Success!
```
